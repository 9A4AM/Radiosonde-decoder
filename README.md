# Radiosonde-decoder
Radiosonde decoder with Python and RS1729 decoder. Not need SOX or Virtual cable. Support RS41, M10, M20 and DFM radiosonde. Only RS41 sonde was logged, because Only RS41 have Payload_ID for now.
